# Cloud Security Lab

Multi-cloud security lab for:

- GCP, AWS, Azure
- Identity-based attack paths
- Sensitive data exposure
- Serverless & AI security
- Palo Alto firewall learning

## Goals

- Demonstrate real-world cloud misconfigurations
- Show identity vs network security differences
- Provide pen testing scenarios
- Enable hands-on learning across clouds

## Structure

- `infra/` → Terraform code
- `scripts/` → start/stop and control scripts
- `docs/` → diagrams, attack paths, notes
